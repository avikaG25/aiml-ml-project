# Student Score Prediction using Machine Learning

##  Description
This project is a simple Machine Learning model that predicts student marks based on the number of hours studied. It uses Linear Regression, a supervised learning algorithm.

##  Objective
To build a model that can predict marks using study hours as input.

##  Technologies Used
- Python 3.x
- Pandas
- Scikit-learn

##  Project Structure
- main.py → Main Python file to run the project
- data.csv → Dataset containing study hours and marks
- requirements.txt → Required libraries
- README.md → Project documentation
